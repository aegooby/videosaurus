# videosaurus &middot;
A saurus for sauruses.
## Initial configuration
### MacOS
Install Homebrew:

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
Install NodeJS and Yarn:

    brew install node
    npm i -g yarn

## Running
Initialize:

    yarn setup
Clean/build:

    yarn clean
    yarn build
Run:

    yarn start
